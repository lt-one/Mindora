# Mindora 功能特性详解

## 1. 个人项目展示

### 功能描述
项目展示模块是个人作品集的展示窗口，以卡片式布局呈现个人开发的各类项目，包含项目缩略图、简介、使用技术栈和链接等信息。

### 核心特性
- 项目卡片展示，包含缩略图、标题和简介
- 按类别筛选项目
- 项目详情页面，展示完整项目描述和演示
- 技术栈标签展示
- 链接到GitHub仓库和在线演示

### 实现技术
- 响应式卡片布局（Tailwind CSS Grid)
- 动态路由（Next.js动态路由）
- 平滑过渡动画（Framer Motion）

## 2. 技术博客

### 功能描述
技术博客模块提供技术文章的发布和阅读功能，支持Markdown格式，并具备分类、标签和搜索功能。

### 核心特性
- 支持Markdown格式的文章编写和渲染
- 文章分类和标签系统
- 代码高亮显示
- 相关文章推荐
- 评论功能（可选）
- 文章搜索和筛选

### 实现技术
- Markdown渲染（react-markdown）
- 代码高亮（rehype-highlight）
- 数据存储（Prisma ORM和MySQL）
- 静态页面生成（Next.js SSG）

## 3. 数据仪表盘

### 功能描述
数据仪表盘模块展示数据可视化能力，通过各类图表展示金融、项目进度或其他数据的统计和分析结果。

### 核心特性
- 多种图表类型（折线图、柱状图、饼图等）
- 实时数据刷新
- 数据范围和维度筛选
- 响应式设计，适配不同设备
- 数据下载和分享功能

### 实现技术
- 数据可视化库（D3.js、Chart.js或Recharts）
- 数据获取和缓存（SWR或React Query）
- 自定义Hook进行数据处理
- 服务器组件进行数据预取

## 4. Todo应用

### 功能描述
Todo应用是一个功能完善的任务管理工具，支持任务创建、编辑、分类、优先级设置和状态跟踪等功能。

### 核心特性
- 任务创建、编辑和删除
- 任务状态管理（待完成、进行中、已完成）
- 任务分类和标签
- 优先级设置和截止日期
- 拖拽排序功能
- 任务筛选和搜索

### 实现技术
- 客户端状态管理（Zustand）
- 拖拽排序（react-beautiful-dnd）
- 日期选择（React Datepicker）
- 本地存储和数据同步
- 服务器操作（Next.js Server Actions）

## 5. 好站分享

### 功能描述
好站分享模块收集和展示高质量的网站资源，按照不同类别进行分类，提供筛选、评分和使用建议等功能。

### 核心特性
- 网站卡片展示，包含预览图、评分和简介
- 按类别和标签筛选
- 详情页展示完整信息和使用技巧
- 用户评分和评论（可选）
- 相关网站推荐

### 实现技术
- 卡片网格布局（CSS Grid）
- 图片懒加载和优化
- 搜索和筛选逻辑
- 动态路由和页面生成

## 6. 通用功能

### 响应式设计
- 适配手机、平板和桌面设备
- 移动优先的开发方式
- 断点优化的布局调整

### 深色模式
- 支持浅色/深色主题切换
- 记住用户偏好设置
- 符合系统主题设置

### 性能优化
- 图片优化和懒加载
- 组件代码分割
- 路由预加载
- 缓存策略

### 无障碍支持
- 语义化HTML结构
- ARIA属性支持
- 键盘导航支持
- 对比度符合WCAG标准
